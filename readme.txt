Shortcodes included with Marketer WordPress theme

== Description ==

This plugin adds shortcode functionality to the Marketer WordPress theme. Having this in it's own plugin means that you can move your content easily if you ever want to change theme. 

Please see [Marketer live demo](http://themersclub.com/marketer/shortcodes) for shortcode usage

Download [WordPress Marketer theme](http://wordpress.org/themes/marketer)

== Installation ==

1. Upload `marketer-addons` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to widgets or add via shortcode
